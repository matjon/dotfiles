To install:

        git clone
        git submodule init
        git submodule update
        cd vim
        wget https://languagetool.org/download/LanguageTool-2.8.zip
        unzip LanguageTool-2.8.zip
        rm LanguageTool-2.8.zip

Then:

- add relevant symlinks to the home dir,

- run vim,

- execute :PluginInstall,

- restart vim.
